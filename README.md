# otterdays.github.io

Static dev profile site, served on [GitHub Pages](https://pages.github.com/).

**Live:** https://otterdays.github.io/

**Features:** Theme switcher (Dark, Light, Google, OpenAI, Anthropic, Otterdays), project badges (Game/Software/Minecraft/Music/Android).

## Local

- Open `index.html` in a browser, or run a static server: `npx serve .`
- Optional: `node fetch-github-repos.js` to generate a repos list (no tokens in repo)

## Repo

- [GitHub](https://github.com/Otterdays/otterdays.github.io)

No build step. No sensitive data in this repo.
